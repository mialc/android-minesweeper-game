# Minesweeper Android App
✅ Java | ✅ Android Studio | ✅ MVVM/MVC | ✅ Dynamic Grid UI | ✅ Fragments

## Features
- Fully functional Minesweeper logic
- Dynamic board generation
- Customizable board size, mine % and tile colors
- Flagging, reveal, flood expansion, win/lose detection
- Restart and multi-screen navigation

## Tech used
- Java
- Android Fragments
- ViewModel / LiveData
- GridLayout programmatic generation
